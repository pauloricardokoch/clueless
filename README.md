# Clueless
Clueless is a simple app to help you when you are out of clue sheets.

You can try it out here [Clueless]

## Usage:

### Setup
```
make setup
```

### Run
```
make run
```

### Clean
```
make clean
```
 
 See logs
 ```
 make logs
 ```
